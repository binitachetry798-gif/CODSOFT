# Simple Rule-Based Chatbot
### CodSoft AI Internship – Task 1

This is a simple chatbot made using Python.  
It gives responses based on basic rules using if-else statements.

The chatbot can:
- Respond to greetings,thanks and goodbye
- Reply to simple questions    
- End the chat when the user types "goodbye"

## How to Run
1. Make sure Python is installed on your laptop.
2. Open the folder named SIMPLE_CHATBOT.PY1.
3. Open a terminal in that folder.
4. Type this command and press Enter:
python Chatbot.py

Now you can chat with the bot!
